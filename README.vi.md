## Tổng quan

Mobile Action Pro là ứng dụng Android đồng hành cho học viên muốn biến kiến thức Kotlin và Jetpack Compose thành thói quen luyện tập mỗi ngày. Ứng dụng hỗ trợ học viên Kotlin Accelerator bằng quiz ngắn, phản hồi tiến độ, hồ sơ cá nhân, bảng xếp hạng và trải nghiệm luyện tập trực tiếp trên điện thoại.

Repo public này chỉ chứa phần giới thiệu sản phẩm, screenshots, logo và các bản APK public release. Source code ứng dụng và hạ tầng khóa học riêng tư không được công khai tại đây.

## Tải xuống

APK public mới nhất:

[Tải APK Mobile Action Pro](https://github.com/dan-tech-academy/mobile-action-pro-public/releases/latest)

Tên file release hiện tại:

`mobile-action-pro-v1.0.0.apk`

## Cài đặt

1. Mở trang release mới nhất của repo này.
2. Tải file APK.
3. Cài đặt trên Android 8.0 trở lên.
4. Nếu Android chặn cài đặt, cho phép cài từ trình duyệt hoặc trình quản lý file rồi thử lại.
